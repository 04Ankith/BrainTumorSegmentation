Brain Tumor Segmentation

Project Overview

This project aims to segment different tumor regions (Enhancing, Non-Enhancing, FLAIR) from MRI scans using a deep learning-based UNet model with hybrid blocks.

Problem

Accurate brain tumor detection from MRI scans is a complex and time-consuming process for radiologists.

Solution

This tool provides automatic segmentation of tumor regions from NIfTI MRI files, saving time and increasing diagnostic support.

\

\

  • Collect and preprocess BraTS dataset
  • Train a hybrid U-Net model with attention and transformer blocks
  • Deploy trained model using Flask
